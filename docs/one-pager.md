# My 0to1 (기존: Founder Discovery) — One-Pager

## THE VISION

1인/초기 창업자가 "세상에서 풀 나만의 문제와 고객", "이들의 문제를 해결해 줄 Fit한 솔루션"을 전담 AI 코치 팀이 함께 찾도록 돕는 **개인용 비즈니스 0to1 운영체제**.

단순 아이디어를 추천해 주는 도구가 아니라, **나 자신에 대한 이해하고, 시장의 문제를 탐색하고, 나와 잘 맞는 문제를 골라 솔루션 핏 검증까지 이어주는 비즈니스 0to1 Copilot**이다.

---

## 00 — 배경

AI와 도구를 활용해서 마음 먹으면 뭐든 '뚝딱' 만들 수 있는 시대다. 하지만 동시에 내가 무엇을 왜 만들고 그것을 지속가능한 사업의 영역으로 이끌어 가는 건 또 다른 문제고 영역이다.

## 01 — PROBLEM

좋은 아이디어를 찾는 단계에서 창업자는 두 가지 함정에 차례로 빠진다.
**함정 1** — 솔루션에 끌려가는 함정. "문제부터 보라"는 말은 알지만 실제로는 트위터·뉴스·사례에서 멋져 보이는 솔루션이 머리를 채우고, 내가 진짜 풀고 싶은 문제와 그냥 매력적인 문제를 구분하기 어렵다.
**함정 2** — 문제 공감을 지불 의사로 착각하는 함정. 인터뷰에서 "맞아요, 저도 그래요"를 듣고 검증된 줄 알지만, 정작 출시하면 아무도 돈을 안 낸다. 인터뷰는 문제의 존재는 검증해도 지불할 만한 고통의 크기는 검증하지 못한다.

---

## 02 — CUSTOMER

**문제/고객 발견을 혼자 붙들고 있는 창업·창업 지향 개인**.

- 1인 창업자, PM 출신 솔로 창업 준비자, 프리랜서/컨설턴트, 커리어 전환을 고민하는 지식 노동자
- 공통점:
    - "나는 뭔가 만들고 싶은데, 무엇을 만들지는 아직 흐릿하다."
    - 스스로와 시장에 대한 이해를 동시에 넓혀야 하는 단계.
    - 여러 도구/템플릿/책은 있지만, **자기 상황에 맞게 이걸 함께 엮어줄 코치/시스템**이 없다.

---

## 03 — CORE MECHANISM

Self map · Problem universe · Founder-problem fit · Validation backlog

이 네 가지가 **끊임없이 순환하며 정교해지는 개인용 발견 시스템**

1. Self map — '나'를 인터뷰하는 Self Insight Agent
    - Agent가 질문을 던져, 관심사·강점·혐오·몰입 경험·경력·접근 가능한 네트워크 등을 인터뷰 형식으로 자연스럽게 끌어낸다.
    - Agent는 고객과의 대화 내용, 답변을 정리해 "나는 어떤 문제에 founder fit이 있는가"를 판단하는 베이스 맵을 만들고 이를 지속적으로 업데이트한다.
2. Problem universe — Seed 스타트업 중심의 문제 지도
    - "세상의 모든 문제"가 아니라, **최근 Seed~Series A 국내외 스타트업이 실제로 풀고 있는 문제**를 중심으로 문제 카드를 수집한다
    - 각 카드에는 누가/언제/왜 겪는지, 현재 대체재가 무엇인지, 무엇이 불편한지까지 같이 기록된다.
3. Founder-problem fit — 문제·사례 후보 큐레이션 + 셀프 평가
    - Self map을 기반으로 Agent가 Problem universe에서 Mina님과 잠재적 fit이 높아 보이는 문제/사례를 추천한다.
    - Mina님이 각 문제에 대해 끌림·이해도·접근성·장기 동기 등을 직접 점수화하고, Agent는 이 피드백을 기반으로 우선순위 상위 1~3개 문제를 정리한다.
4. Validation backlog — 사업/서비스 아이디어 초안 + 검증 플랜
    - 우선순위가 높은 문제에 대해 Agent가
        - 이 문제가 진짜 사업이 될지를 가르는 4가지 가설(문제 존재 여부 / 심각도 / 솔루션 핏 / 지불 의사)을 각각 가장 싸고 빠르게 깨는 검증 가이드를 제공한다.
        - 가설마다 인터뷰·관찰·스모크 테스트·페이크 도어·소액 선결제·컨시어지 PoC 중 적합한 방법이 처방된다.

이 네 단계는 일회성이 아니라, **서로 연결되어 반복되는 루프**로 설계되어 UX적으로도 seamless하게 연결되어야 한다.

### 이 네 단계의 시스템이 일반 LLM과 다른 이유 — Real Moat
이 시스템의 진짜 moat은 누적되는 컨텍스트의 깊이다. Self Map에 쌓이는 본인의 강점·혐오·네트워크, Problem Universe에서 본 시장 케이스, 그동안 깨본 가설과 그 결과 — 이 모든 것이 누적될수록, 같은 "이 가설 약한 것 같은데요"라는 코멘트도 ChatGPT가 백지에서 내놓는 그것과는 정확도와 설득력이 완전히 달라진다. 일반 LLM 도구가 모방할 수 없는 지점이 여기다.

## 04 — AGENT ROLES

**각 단계에 특화된 전담 에이전트**를 둔다.

● Self Insight Agent
    - Mina님에게 관련한 의미있는 질문을 지속적으로 던져 자기 이해를 구조화하고, 메타 인지를 강화하며, Self map DB를 지속적으로 업데이트한다.
● Problem Scout Agent
    - 국내외 Seed 투자 스타트업·시장 기사·커뮤니티 등을 기반으로 Problem universe 후보를 수집하고, 문제·고객·현재 대체재를 요약한다
● Fit Judge Agent
    - Self map과 Problem universe를 비교해 "fit 있어 보이는 문제/사례"를 추천 리스트로 만들고, Mina님의 셀프 평가를 돕는다.
● Validation Design & Coach Agent [핵심]
    - 누적된 유저의 컨텍스트(Self Map / 과거 인터뷰 결과 / 이전에 깬 가설들) 위에서, 적합한 한 걸음을 처방한다
    - 이 문제가 진짜 사업(o to1)이 될지를 가르는 4가지 핵심 가설(문제 존재 여부 / 심각도 / 솔루션 핏 / 지불 의사)을 각각 가장 싸고 빠르게 깨는 검증 가이드를 제공한다.
    - "누구를 대상으로" "무엇을 어떻게 검증할지" 인터뷰 질문이나 설문, 실험 설계 및 방법, 성공/실패 시그널 등 1pager 기반으로 구체적이고도 명확하게 설계해 제공한다.
● Reality Check Agent
    - "이건 창업자의 희망사항일 뿐일 수 있다", "솔루션이 먼저인 것 같다" 등 편향을 지적하고, 더 필요한 증거를 요구한다.

---

## 05 — WHY THIS MATTERS

이 시스템은 "아이디어 생성기"가 아니라, **창업자의 시간과 에너지를 가장 가치 있는 문제로 집중시키고 검증까지 실행하게 하는 맞춤형 창업 보조 장치**다.

- 창업자는 더 이상 무한한 아이디어/문제 목록 속에서 헤매지 않고,
    - 자신과 진짜 맞는 문제,
    - 이미 시장에서 의미 있게 다뤄지고 있는 문제,
    - 실제 검증 가능한 문제에 대해 
    본인만의 우선순위를 매기고 집중할 수 있다.

- AI Agent는 "생각을 대신"하는 것이 아니라,
    - 생각의 구조를 잡고,
    - 자료 수집과 비교를 돕고,
    - 편향을 짚어주며,
    - 다음 검증 스텝을 제안하는 **디스커버리 파트너**가 된다

- 결과적으로, 무엇을 할지 고민하는 시간은 줄이고, 이 문제를 풀어도 되는지에 대한 검증된 확신을 쌓는다 — 단순한 공감이 아니라, 시장이 돈을 낼 만한 고통을 발견할 때까지.
